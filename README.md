# Fiona Feats Familiar Tracker

### Checklist for updates
- look up & note down sources of all familiars
- check the date in FR time
- update bio ver
  - note down where to paste after
  - remember to update version number
  - quickly test on a dragon
  - copy over to `/Template/bio-version.txt`
- update forum ver in example post
  - remember to update version number
  - fix spacing, take note of where
  - copy over to `/Template/forum-version.txt`
- make update post with `/Tools/update-post.py`
- add new instructions to archive (& save to git)
- update changelog in main post (& save to git)
- update personal tracker using instructions
